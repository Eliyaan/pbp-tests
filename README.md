## PBP Tests

Prerequisite: have [V](https://github.com/vlang/v) installed

To run the project : 
- install [Eliyaan/pbp-editor](https://github.com/Eliyaan/pbp-editor) (and compile it)
- clone this repo
- run pbp-editor in this repo's folder (and put the result in a `main.v` file (`pbp-editor >main.v`))
- link the Parts
- close pbp-editor
- run `v run .` in this repo's folder
